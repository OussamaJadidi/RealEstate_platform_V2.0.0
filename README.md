# Project Description

The project is a responsive Real-Estate web application, 
The revolution platform for Real Estate, 
    Buy, Rent or Sell your property can't be more easy with Estate Elite,  

## Main Technologies that i used are

- Next.js
- Prisma (ORM)
- tailwind css
